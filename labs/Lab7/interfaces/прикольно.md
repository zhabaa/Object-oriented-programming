# Скриптинг баш

Создание файлов с похожими именами используя лишь одну команду

```bash 
for i in {1..3}; do code "interface${i}.py"; done
```

Более общий вариант:

```bash
n=7 && suff="json" && name="name" && for i in {1..$n}; do touch "${name}${i}.${suff}"; done
```

Вместо `touch` можно использовать любую другую команду.
